# Student Grade Calculator

## Description
A Java console application that calculates the total marks, average percentage, and grade of a student based on marks entered for multiple subjects.

## Features
- Accept marks for multiple subjects
- Calculate total marks
- Calculate average percentage
- Assign grades automatically
- Validate input marks (0-100)

## Technologies Used
- Java
- Scanner Class
- Loops
- Conditional Statements

## Sample Input
Enter the number of subjects: 5

Enter marks for Subject 1: 90
Enter marks for Subject 2: 85
Enter marks for Subject 3: 78
Enter marks for Subject 4: 92
Enter marks for Subject 5: 88

## Sample Output
Total Marks = 433
Average Percentage = 86.6%
Grade = B
